# What Happens When You Type google.com in Your Browser and Press Enter

## 📌 Project Overview
This project explains what happens behind the scenes when a user types **https://www.google.com** into a browser and presses Enter.

It covers the full journey of a web request, including networking, security, servers, and browser rendering.

## 🔍 Topics Covered

- URL Parsing
- DNS Resolution
- TCP/IP (3-way handshake)
- Firewall
- HTTPS / TLS Encryption
- Load Balancer
- Web Server / Edge Server
- Application Server
- Database / Cache
- Browser Rendering

## 🎯 Objective

The goal of this project is to understand how the web works end-to-end and to demonstrate the ability to explain complex systems clearly.
